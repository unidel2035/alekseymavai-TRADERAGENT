# TRADERAGENT - TradingView to Bybit Automated Trading System

Автоматизированная торговая система, которая использует сигналы индикаторов TradingView для размещения ордеров на бирже Bybit через API.

## Возможности

- Анализ рынка с использованием технических индикаторов (RSI, MACD, Bollinger Bands, EMA)
- Автоматическая генерация торговых сигналов
- Визуализация сигналов и уровней SL/TP на графике TradingView
- Автоматическое размещение ордеров на Bybit через webhook
- Управление рисками с настраиваемыми параметрами
- Поддержка testnet для безопасного тестирования

## Компоненты системы

1. **Pine Script индикатор** (`pine_script_indicator.pine`) - анализирует рынок и генерирует сигналы
2. **Webhook сервер** (`webhook_server/`) - принимает сигналы от TradingView и отправляет ордера на Bybit
3. **Документация** (`docs/`) - инструкции по установке и использованию

## Быстрый старт

1. Установите индикатор в TradingView
2. Настройте Bybit API ключи
3. Запустите webhook сервер
4. Создайте алерт в TradingView с webhook URL

Подробные инструкции в [docs/setup.md](docs/setup.md)

## Требования

- TradingView Pro/Pro+/Premium аккаунт (для webhook функциональности)
- Bybit аккаунт с API доступом
- Node.js 14+ для webhook сервера
- Сервер с публичным IP (или ngrok для тестирования)
